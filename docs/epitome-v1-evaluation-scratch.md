### Specific Issues Identified

#### 1. Critical Apparatus Notes (SEVERE - ~30% of paragraphs)
The following patterns are NOT being filtered:
- Manuscript sigla followed by variant readings: `"δ᾽ Α."`, `"À."`, `"PW"`, `"B."`
- `codex`/`codices` references: `"alter cod. Wolfii"`, `"codex Colberteus"`
- `om` (omit) markers: `"om A"`, `"om ἃ"`
- `add` markers: `"A add ὁ"`, `"post μυριάδες Α add δ᾽"`
- Square bracket notation: `"τῶν ἀστέρων] ἀστέρας Α"`
- Editor names: `"Wolfius"`, `"Ducangius"`, `"Iosephus"`

#### 2. Editor/Scholar References (MODERATE - ~10%)
- `"Theodoretus p. 1173"`, `"Theodoreto"`
- `"Plutarchus tantur"`
- `"Hoeschelium (H)"`
- `"Ioannis Damasceni"`
- Page/line references: `"p. 229 v. 13"`, `"p. 1195 Τὸ 1201"`

#### 3. Latin Text Leakage (MODERATE - ~5%)
- `"scilicet Daniel 5 30"`
- `"apud Hoeschelium"`
- `"nonnullis mutatis"`
- `"tuendum sit"`
- `"ex Wolfii coniectura prae- staret"`
- `"modo sic modo"`

#### 4. Fragment/Stub Paragraphs
Several single-word or incomplete-sentence paragraphs survive:
- `"χρησάμενος."`
- `"διαφϑείραντος."`
- `"ἐνιαυτός.\" à;"`
- `"TUSES —"`

#### 5. OCR Artifacts
- `"slg"` in index 0 (should be Greek)
- `"9:07"` in index 20 (garbled)

---

### Recommendations for Cleaning Agent

#### HIGH PRIORITY - Must fix for B+ grade:

1. **Filter paragraphs containing manuscript sigla patterns:**
   - Regex: `\b[ABCDEPWR]\.\s` or `\bÀ\.\s` (capital letter + period + space)
   - Regex: `alter cod\.` or `codex \w+`
   - Regex: `\bom\s+[A-Z]` (omit markers)
   - Regex: `\badd\s+['"ὁτό]` (add markers)
   - Regex: `\]\s*\w+\s+[A-Z]` (square bracket apparatus)

2. **Filter paragraphs with editor/scholar names:**
   - Match: `Wolfii`, `Wolfius`, `Ducang`, `Theodoret`, `Plutarch`, `Ioseph`, `Hoeschel`
   - Match: `codex Colbert`, `codex Reg`

3. **Filter paragraphs with Latin apparatus phrases:**
   - Match: `scilicet`, `apud`, `tantur`, `tantum`, `nonnullis`, `mutatis`, `tuendum`
   - Match: `modo sic modo`
   - Match: `ex \w+ coniectura`
   - Match: `p\.\s*\d+` (page references)
   - Match: `v\.\s*\d+` (verse/line references)

4. **Filter very short paragraphs (<30 characters) that are likely fragments**

5. **Merge adjacent fragments into preceding/following paragraphs where coherent**

#### MEDIUM PRIORITY:

6. **Strip embedded line numbers from valid Greek text**
   - Pattern: isolated digits at start/middle of paragraphs

7. **Fix OCR artifacts within otherwise valid paragraphs**
   - `"slg"` -> `"τὴν"`

---

### Verdict

**NEEDS MORE WORK** - Current output is unsuitable for translation.

The cleaning script is removing some FONTES citations but is NOT catching the vast majority of critical apparatus contamination. Nearly every chapter has dozens of apparatus-only paragraphs that should be stripped entirely.

The Cleaning Agent must implement the regex patterns above to filter out:
- Manuscript variant readings (A, B, PW, etc.)
- Editor references (Wolfius, Ducangius, etc.)
- Latin scholarly notes
- Page/line references

Once these patterns are filtered, the output should approach B+ quality.
